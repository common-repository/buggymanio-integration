== Description ==

Plugin for integration with Buggyman.io tool for monitoring of your PHP & Javascript errors and exceptions.
Note: this plugin connects to an external server of Buggyman for error reporting using CURL

== Installation ==

0. Register an account at [Buggyman.io](http://buggyman.io/?utm_source=wordpress_repository).
1. Upload `buggyman-wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set PHP & JS token keys at plugin options page
